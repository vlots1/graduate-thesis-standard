# Conflicts and Ambiguities

| ID | Evidence | Treatment | Reason |
| --- | --- | --- | --- |
| C-01 | `R p.5` says figures/tables may be chapter-numbered when numerous; `R pp.8-9` says body figures/tables are continuous to the appendices. Word and supplied LaTeX default to continuous (`L bnusetup.tex:119`). | **Mandatory/manual:** report the discrepancy; default to the more specific printed-format rule (continuous) for a new BNU thesis, but flag an existing chapter-numbered thesis for faculty confirmation rather than auto-renumbering. | Same official source contains both formulations. |
| C-02 | `R p.2` cites GB/T 7714 without edition; `R p.9` and Word template cite GB/T 7714-2015. | **Mandatory/manual:** require GB/T 7714 but do not claim an edition is resolved. Preserve an already approved system; ask the graduate office/faculty before changing edition-specific fields. | 2026 rule internally has a version ambiguity. |
| C-03 | `R p.3` calls appendix “有则必备”; `R p.5` calls it optional. | Treat appendix as optional; once present it must follow appendix rules. | Both statements convey the same conditional requirement. |
| C-04 | `R p.3` lists acknowledgements/postscript/statement; `R pp.6-7` assembly specifically lists acknowledgements, then committee list. | Require the listed assembly for a submission audit; ask if an approved postscript/statement replaces acknowledgements. | Later assembly table is more specific. |
| C-05 | Word template includes instructional prose, sample outlines, placeholder bibliography/index entries, and figure/table how-to content. LaTeX example includes optional algorithms and bachelor/APA/MLA capabilities. | Label all such material **observational** unless independently supported by `R`. | They illustrate use; they do not prescribe thesis content. |
| C-06 | `R p.9` reference-list body is 五号/16 pt while main body and appendix are 小四/20 pt; Word style Normal defaults to 小四 but sample reference paragraphs rely on manual text. | Treat reference layout as mandatory only when the source is clear, but require visual output inspection in DOCX; do not force Normal style onto references. | A real Word document needs paragraph-level inspection. |
