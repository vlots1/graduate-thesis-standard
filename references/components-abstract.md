# Components, Abstracts, and Keywords

## Thesis composition model

Use this order unless an official rule, degree type, or approved faculty requirement changes it. `R p.3` gives the main components; `R pp.6-7` gives the binding print assembly order.

| Component | Status | Numbered / in contents | Evidence and check |
| --- | --- | --- |
| Cover | mandatory | no / no | Title, author, student ID, discipline, supervisor, unit, date. Title normally ≤25 Chinese characters. `R p.3`, `R p.7`. |
| Title page | mandatory | no / no | Same identity fields; English cover corresponds to Chinese cover. `R p.3`. |
| Originality statement and authorization | mandatory | no / no | Use the supplied text/layout; do not rewrite legal content. `R p.3`; `W paras 26-41`. |
| Spine | doctoral mandatory; master optional | no / no | `R p.7`; `W para 48`; `L bnusetup.tex:86-91`. |
| Chinese and foreign-language abstracts | mandatory | front matter / no | Content correspondence is mandatory. `R pp.3-4`. |
| Keywords | mandatory | associated with abstracts / no | 3–8; corresponding English terms. `R p.4`. |
| Contents | mandatory | front matter / itself no | Starts listing from introduction/preface; list chapter, section, item and start pages. `R pp.4,8`. |
| Figure/table lists | conditional | front matter / no | Required when figures/tables are numerous; separate lists. `R p.4`. |
| Main symbols list | conditional | front matter / no | Required when symbols/abbreviations etc. are numerous; otherwise explain on first use. `R p.4`. |
| Main text | mandatory | numbered / yes | Includes introduction and conclusion/discussion. `R pp.3-4`. |
| References | mandatory | unnumbered back matter / yes | Begins on a new page after main text. `R p.5`. |
| Appendix | optional, but complete if used | lettered / yes | `R p.6`. |
| Index | optional | unnumbered / yes if included | After appendix; use GB/T 41210 if present. `R p.6`. |
| Degree-period achievements | mandatory | unnumbered / yes | Include papers/other outputs with requested bibliographic fields. `R p.6`. |
| Acknowledgements, postscript, or statement | mandatory in the listed assembly | unnumbered / yes | Factual and sincere. `R pp.3,6-7`. |
| Defence committee list | mandatory in 2026 assembly order | unnumbered / yes | `R pp.6-7`; use table in Word template. |

## Abstract constraints

- **Mandatory:** State purpose, method, result, and conclusion; highlight innovation; be self-contained and without annotation/comment. `R pp.3-4`.
- **Mandatory:** Doctoral Chinese abstract is generally about 2,000 Chinese characters; master's is generally about 1,000. Treat “generally” as an expected range, not an automatic rejection threshold. `R pp.3-4`.
- **Mandatory:** Chinese and English abstracts correspond in content. A thesis mainly written in English needs a detailed Chinese abstract, generally no fewer than 5,000 characters. `R p.4`.
- **Mandatory:** Avoid figures, tables, chemical structures, and non-public symbols/terms in an abstract where possible. `R p.4`.
- **Mandatory layout:** Chinese title / `摘要`; Chinese body in Songti 小四, 20 pt fixed spacing and 2-character first-line indent. English title in uppercase Times New Roman 三号; `ABSTRACT` and body follow the printed layout. `R p.7`.
- **Recommended:** Do not add headings inside an abstract unless the faculty requires them. The school rule does not specify abstract subheadings.

## Keywords

- Select **3–8**, put them on a new line below the abstract, and separate Chinese keywords with Chinese commas. `R p.4`, `R p.7`.
- Put corresponding English keywords below the English abstract; capitalize their first letters and separate them with half-width commas. `R p.7`; `W paras 74-75`.
- Audit correspondence as a semantic/manual check; count and separator style can be automated.
